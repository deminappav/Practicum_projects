### 🔌 Тестирование API приложения «Яндекс.Прилавок»

> **О проекте:**  
> **«Яндекс.Прилавок»** — онлайн-сервис с каталогом товаров. Пользователь может просматривать товары, добавлять их в корзину, оформлять заказы и следить за их статусами.

`API Testing` `REST API` `Postman` `cURL` `JSON` `Bug Reporting` `Backend Testing`

* **Тестирование:** Провела комплексное тестирование API (**122 сценария** для 5 ключевых эндпоинтов).  
  * **Результат:** Полностью оценила надежность серверной части и корректность обработки входящих данных.

* **Работа с дефектами:** Локализовала и задокументировала **59 дефектов API**, оформив детальные баг-репорты с cURL-запросами и ответами сервера.  
  * **Результат:** Защитила бэкенд от нештатных падений, предотвратила ошибки интеграции и повысила стабильность всего сервиса.
