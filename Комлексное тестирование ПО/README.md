### 🎓 Дипломный проект: Комплексное тестирование ПО (Web, Mobile, API)

* **Сквозное тестирование (End-to-End):**  
  Провела полное сквозное тестирование сервиса (**287 проверок**), зафиксировав **88 багов** (из них 14 найдены вне документации).  
  * **Результат:** Обеспечила полную и объективную оценку качества системы перед релизом.

* **Тестирование Web-клиента:**  
  Проверила вёрстку и валидацию (**159 сценариев** для экрана «Статус заказа» и формы оформления).  
  * **Результат:** Предотвратила потерю заказов, локализовав критические блокеры (включая неработающую кнопку в Chrome).

* **Тестирование Mobile-приложения:**  
  Проверила функциональность в курьерском режиме (**72 теста** с имитацией сети 3G, прерываний и смены статусов).  
  * **Результат:** Защитила бизнес от сорванных доставок и финансовых потерь курьеров «в поле».

* **Тестирование API:**  
  Проверила 3 ключевых эндпоинта (**56 тестов** на валидацию и бизнес-логику).  
  * **Результат:** Защитила бэкенд от падений (`500 Internal Server Error`) и рассинхронизации данных в базе данных.
