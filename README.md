# 📂 Учебные проекты QA (Практикум)

В этом репозитории собраны мои проекты по тестированию программного обеспечения, выполненные в ходе обучения. 

### 📁 Структура репозитория

Каждая папка представляет собой отдельный проект со всей рабочей документацией. **Внутри каждой папки есть свой `README.md`** с подробным описанием проделанных работ, метриками проверок и найденных дефектов:

* **`Веб_тестирование/`**  
  Функциональное и регрессионное тестирование веб-приложения (тестирование верстки, пользовательских форм, логики построения маршрутов, чек-листы и баг-репорты).

* **`Мобильное_тестирование/`**  
  Функциональное и регрессионное тестирование мобильного приложения (чек-листы, баг-репорты, проверка пользовательских сценариев).

* **`Тестирование_API/`**  
  Тестирование REST API сервиса (проверка эндпоинтов, cURL-запросы, валидация данных и бизнес-логики).

* **`Дипломный_проект/`**  
  Комплексное сквозное тестирование (End-to-End) системы, включающее Web, Mobile и API части.
  
---
> 💡 *Перейдите в папку интересующего проекта, чтобы ознакомиться с его полным описанием и рабочей документацией.*
